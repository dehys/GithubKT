## Implementation ##

Maven
```xml
<repositories>
  <repository>
    <id>jitpack.io</id>
    <url>https://jitpack.io</url>
  </repository>
</repositories>

<dependencies>
  <dependency>
    <groupId>com.github.dehys</groupId>
    <artifactId>GithubKT</artifactId>
    <version>VERSION</version>
  </dependency>
</dependencies>
```
<br>

Gradle
```groovy
repositories {
  maven {
    url 'https://jitpack.io'
  }
}

dependencies {
  compile 'com.github.dehys:GithubKT:VERSION'
}
```
